Security Layer

RLS & Triggers

Keamanan 4 layer memastikan data terlindungi. Mahasiswa tidak bisa sembarangan mengedit data orang lain.

  • Auth Layer: JWT + Validasi Email @student
  • Database Layer: Row Level Security (RLS) di 5 tabel
  • Admin Ops: RPC Security Definer (bypass RLS secara aman)
  • Automation: PostgreSQL Triggers untuk notifikasi otomatis

Deployment

Cara Install & Build

Aplikasi dapat dijalankan langsung via Expo Go atau diinstal sebagai APK Production yang sudah jadi.

Install via Expo Go

1. Install Expo Go dari App Store / Play Store
2. Scan QR code dari npx expo start
3. App langsung jalan (no build required)

Install APK (Android)

1. Download CariIn-v1.apk (76 MB)
2. Buka file → Install → Izinkan unknown sources
