Realitas saat ini

Grup WhatsApp itu untuk ngobrol.

Bukan untuk nyari kunci motor yang hilang.

Kalau pakai WA:

  • Info ketumpuk chat lain.
  • Privasi (nomor HP) terekspos.
  • Rawan orang ngaku-ngaku (klaim palsu).

Masa depan kok masih manual?

Mahasiswa butuh kepastian. Kehilangan barang bikin panik, sistem yang berantakan bikin makin stres.

Fakta Lapangan

Kita tanya ke 39 anak kampus...

51%

Pernah kehilangan barang di kampus.

67%

Larinya minta tolong ke Satpam.

Jawabannya ada di Cari.In.

Platform terpusat khusus mahasiswa. Aman, terverifikasi, dan langsung hubungi penemu tanpa perlu sebar nomor pribadi.

Kenapa Beda?

1. Khusus Email Kampus

Wajib pakai @student.unu-jogja.ac.id. Identitas 100% valid, gak ada akun bodong.

2. Chat In-App Realtime

Langsung ngobrol di dalam aplikasi. Nomor WhatsApp kamu tetap rahasia.

3. Satpam Punya Akses

Mahasiswa bisa lapor langsung ke pos, satpam yang bantu input ke sistem.

Desain
Bukan Cuma Kosmetik.

Kita pakai pendekatan psikologi warna "Calm Campus".

Blue (Utama)

Menurunkan detak jantung & panik.

Soft Orange

Urgensi, tapi tidak bikin stres seperti merah.

Emerald

Simbol barang ditemukan (Relief).

Mesin di Balik Layar

React Native 0.81 Expo SDK 54 NativeWind v4 Zustand State
Supabase Auth PostgreSQL Realtime WebSockets EAS Cloud Build
Keamanan Database

Data kamu
nggak bisa diintip.

Menggunakan RLS (Row Level Security).

Mahasiswa hanya bisa mengedit laporannya sendiri. Laporan milik orang lain hanya bisa dilihat jika sudah di-approve oleh admin. Admin menggunakan RPC (Remote Procedure Call) untuk intervensi aman.

Instan.

Chat masuk detik itu juga tanpa perlu refresh.

Supabase postgres_changes secara konstan me-listen setiap INSERT pesan baru, langsung memicu notifikasi in-app ke penerima.
