Core Engine

Realtime Communication


// Supabase Realtime Subscription Channel
const channel = supabase
  .channel(`messages:${conversationId}`)
  .on(
    'postgres_changes',
    { event: 'INSERT', schema: 'public', table: 'messages' },
    (payload) => {
      // Chat bubble muncul instant tanpa reload halaman
      addMessageToState(payload.new);
    }
  ).subscribe();

postgres_changes

Listen spesifik pada event INSERT tabel messages secara live websocket.

DB Trigger

Message insert trigger auto-insert notifikasi, update bell badge app.

useFocusEffect

Subscribe channel hanya saat fokus di ChatRoom, memory efficient.
