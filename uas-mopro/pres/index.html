The Problem

Realitas di Lapangan

Mahasiswa kehilangan barang. Grup WhatsApp menjadi sarana utama, namun menimbulkan berbagai masalah operasional dan privasi.

Informasi Tenggelam

Laporan barang hilang di grup WA tertumpuk oleh obrolan lain. Sulit dilacak dan tidak ada sistem pencarian yang terstruktur.

Klaim Palsu

Tanpa sistem verifikasi identitas, siapa saja di grup publik dapat mengaku sebagai pemilik barang yang ditemukan.

Privasi Terancam

Nomor handphone pelapor langsung terekspos ke ratusan orang di grup publik, berpotensi memicu spam atau penyalahgunaan.

Unique Value Proposition

Apa yang Berbeda?

Domain Email

Registrasi hanya menggunakan @student.unu-jogja.ac.id. Identitas terjamin.

Admin Moderation

Laporan difilter (Approve/Reject) sebelum tayang untuk mencegah spam.

In-App Chat

Komunikasi realtime aman tanpa perlu bertukar nomor WhatsApp.

Titik Penitipan

Informasi lokasi fisik (Pos Satpam/Fakultas) tempat barang diamankan.

Visual Intuitif

Orange untuk label Hilang, Emerald untuk Ditemukan. Mata cepat mengenali.

Walk-in Report

Fitur bagi satpam untuk input manual jika mahasiswa lapor langsung secara fisik.
